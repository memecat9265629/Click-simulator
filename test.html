iOS IPA Installation

Choose a method to install IPA files on your iOS device.

ESign (No Jailbreak)

ESign is a free IPA signature tool that allows users to install IPA files without a PC.

Download ESign IPA Direct Install

AltStore (PC Required)

AltStore lets you install and sign IPA files using a computer and Wi-Fi.

Download AltStore

Scarlet (No Jailbreak)

Scarlet is an alternative app store that allows you to install IPA files directly on iOS.

Download Scarlet

TrollInstallerX (iOS 15-16)

TrollInstallerX enables app sideloading on iOS 15+ without a computer.

Download TrollInstallerX

Jailbreak (Advanced Users)

Jailbreaking unlocks full system access, allowing you to install apps without restrictions.

Learn More About Jailbreaking

Certificate for Signing

To install and sign apps, you need a valid certificate. Download and install it below:

Download Certificate

After downloading, go to **Settings > Profile Downloaded** to install.

How to Install & Trust Apps

  • Download the IPA file or install directly.
  • Install the certificate and go to **Settings > General > Profiles & Device Management**.
  • Find the certificate, tap **Trust**, and confirm.
  • Now you can open the installed app.
